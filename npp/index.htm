如何在Notepad++中直接執行Python或Go的程式

step1. 點主選單"外掛"→"外掛模組管理"

step2. 搜尋 "NppExec" 然後將它勾選 再按 [安裝]

step3. 按 [是]

step4. 重啟Notepad++ 按 [F6] 開啟 Execute... 視窗 在裡面輸入 python "$(FULL_CURRENT_PATH) 然後按 [Save...]

step5. 在Script name:裡輸入 Python 然後按 [Save]

step6. 在Notepad++裡面編輯好python文件存檔 按 [F6] 確定Script name是 Python 然後點 [ok] 就會在編輯區下方看到cmd執行結果
